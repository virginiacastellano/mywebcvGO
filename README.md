## Estructura del Proyecto

El proyecto está organizado de la siguiente manera:

- **main.go:** Archivo principal que inicia el servidor web Go.
- **static/:** Directorio que contiene archivos estáticos, como estilos CSS e imágenes.
- **templates/:** Directorio que contiene los archivos HTML de las páginas web.

## Tecnologías Utilizadas

- **Go:** Lenguaje de programación principal.
- **HTML:** Utilizado para crear la estructura del currículum vitae.
- **Bootstrap:** Framework de CSS y JavaScript para mejorar el diseño y la experiencia del usuario.

## Instrucciones de Ejecución

1. Clona el repositorio:

   ```bash
   git clone https://github.com/virginiacastellano/mywebcvGO.git
   ```

2. Navega al directorio del proyecto:

   ```bash
   cd mywebcvGO
   ```

3. Ejecuta el servidor web Go:

   ```bash
   go run app.go
   ```
